Responsive Design

🎯 What Responsive Columns Do:
Responsive columns automatically adjust their width and layout based on screen size. For example, col-12 col-md-6 col-lg-4 means: • Mobile (XS): 12 columns wide (full width) - stacked vertically
• Tablet (MD): 6 columns wide (half width) - 2 per row
• Desktop (LG): 4 columns wide (third width) - 3 per row
Try resizing your browser window to see this in action!

Responsive Grid

How Responsive Grids Work:
• XS (0-575px): All columns stack vertically (full width)
• SM (576-767px): Columns can be 6 wide (2 per row)
• MD (768-991px): Columns can be 4 wide (3 per row)
• LG (992-1199px): Columns can be 3 wide (4 per row)
• XL (1200-1399px): Columns can be 2 wide (6 per row)
• XXL (1400px+): Columns can be 2 wide (6 per row)
